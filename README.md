# Tanks Game

Browser tank arena built with TypeScript, Vite, Canvas 2D, and a small Node WebSocket server.

## Setup

Install root frontend tooling:

```sh
npm install
```

Install server dependencies:

```sh
cd server
npm install
```

## Development

Run the WebSocket server:

```sh
npm run server
```

Run the Vite client in another terminal:

```sh
npm run dev
```

Open `http://localhost:8080`.

## Build

```sh
npm run build
```
